🌍 1. პროფესიონალური დამიწების კონტურის გაანგარიშება

✓ გაანგარიშების დეტალური შედეგი:

📘 დამიწების სისტემები: TN-S, TN-C-S და TT

ელექტროუსაფრთხოების მთავარი პრინციპია ფაზური სადენის კორპუსზე შეხებისას მოკლე შერთვის დენის წარმოქმნა, რაც მყისიერად თიშავს ავტომატურ ამომრთველს ან დიფერენციალურ რელეს (RCD).

  • TN-C-S: PEN სადენი იყოფა PE (დამიწება) და N (ნული) გამტარებად შენობის მთავარ შესასვლელ ფარში. აუცილებელია განმეორებითი დამიწების კონტური ($\le 10\ \Omega$).
  • TT სისტემა: შენობის დამიწება სრულად დამოუკიდებელია ქვესადგურის ნულისგან. სავალდებულოა RCD დაცვა! გამოიყენება სოფლის დასახლებებში, სადაც საჰაერო ხაზის დაზიანების რისკია.

🛠️ პოტენციალთა გათანაბრების მთავარი შინა (GSH / GSU)

შენობის შიგნით ყველა მეტალის კონსტრუქცია (წყლის მილები, გაზის მილები, არმატურა, ლიფტის შახტა, მეხამრიდი) უნდა შეერთდეს ერთიან სპილენძის ან ფოლადის სალტეზე (GSH). ეს გამორიცხავს ორ სხვადასხვა საგანს შორის საშიში ძაბვათა სხვაობის წარმოქმნას.

📐 მასალების მინიმალური კვეთები (GOST / PUE):

  • ვერტიკალური ფოლადი: Ø $\ge 16$ მმ (მოთუთიებული) ან $50\times50\times5$ მმ კუთხოვანა.
  • ჰორიზონტალური ზოლი (შლეიფი): ფოლადის ზოლი $40\times4$ მმ ან Ø $12$ მმ კატანკა.
  • მთავარი დამიწების გამტარი ფარიდან კონტურამდე: სპილენძი $\ge 10\text{ მმ}^2$, ალუმინი $\ge 16\text{ მმ}^2$, ფოლადი $\ge 75\text{ მმ}^2$.

🔥 შედუღება და ანტიკოროზიული დამუშავება:

მიწისქვეშა ყველა გადაბმა უნდა შესრულდეს მხოლოდ ნაკეროვანი შედუღებით (შედუღების სიგრძე $\ge 2 \times \text{ზოლის სიგანე}$). შედუღების ადგილი უნდა დაიფაროს ბიტუმით. ყურადღება: თავად ღეროებს საღებავი არ უნდა წაესვას!

🛡️ 2. მეხამრიდის & იმპულსური გადაძაბვის დაცვა (IEC 62305 / IEC 61643)

✓ გაანგარიშების შედეგი:

⚡ 1. გარე მეხთამრიდობა (LPS) და ჩამოშვიდი გამტარები

ჩამომშვიდი გამტარები (Down Conductors) პასუხისმგებელია მეხის დენის ($I_{imp} \ge 100\text{ kA}$) უსაფრთხო გატარებაზე სახურავიდან მიწამდე.

  • მანძილი ჩამომშვებებს შორის: Class I: 10მ, Class II: 15მ, Class III: 20მ.
  • გამტარის მასალა: მოთუთიებული ფოლადი Ø $8$ მმ, ალუმინი Ø $8$ მმ, ან სპილენძი $\ge 50\text{ მმ}^2$.
  • მოსახვევის რადიუსი: მინიმუმ $20$ სმ, არ შეიძლება მახვილი ($90^\circ$) კუთხეები!

🛡️ 2. შიდა დაცვა — SPD / УЗИП აპარატები (IEC 61643)

პირდაპირი მეხის დარტყმა იწვევს ქსელში ათასობით ვოლტის იმპულსს. ამისგან ტექნიკის გადასარჩენად გამოიყენება 3-საფეხურიანი დაცვა:

  • Type 1 (Class I): დგება მთავარ შესასვლელ ფარში. უძლებს პირდაპირ მეხის დარტყმას ($10/350\ \mu s$ ტალღა).
  • Type 2 (Class II): დგება სართულებრივ/განამაწილებელ ფარებში. იცავს ინდუქციური და ქსელური გადართვებისგან ($8/20\ \mu s$ ტალღა).
  • Type 3 (Class III): დგება უშუალოდ მგრძნობიარე ტექნიკასთან (ქვაბი, კომპიუტერი, სამედიცინო აპარატურა).

🌐 3. მგორავი სფეროს მეთოდი (Rolling Sphere Method):

წარმოიდგინეთ $R$ რადიუსის მქონე სფერო (Class III-სთვის $R=45$მ), რომელსაც დავაგორებთ შენობაზე. ყველა წერტილი, რომელსაც სფერო ფიზიკურად შეეხება, საჭიროებს მეხმიმღებს. სივრცე სფეროს ქვეშ ითვლება აბსოლუტურად დაცულად.

☀️ 3. მზის პანელები და ავტონომიური სისტემები

🧩 პანელების ტიპები

• მონოკრისტალური (Mono): მაღალი მარგი ქმედების კოეფიციენტი (20-23%), საუკეთესო ეფექტურობა.

• პოლიკრისტალური (Poly): ბიუჯეტური, შედარებით დაბალი მარგი ქმედებით (15-17%).

⚙️ სისტემის კომპონენტები

• ინვერტორი (Hybrid/Off-Grid): DC დენის გადაყვანა 220V AC-ში.

• MPPT კონტროლერი: მზის ენერგიის მაქსიმალური ათვისება და აკუმულატორის დატენვა.

🛡️ დაცვის საშუალებები

• DC SPD (УЗИП): მეხისგან და იმპულსური გადაძაბვისგან დაცვა პანელების ხაზზე.

• DC ავტომატები & მცველები: სპეციალური მუდმივი დენის ავტომატები.

🧮 მზის ელექტროსადგურის ჭკვიანი კალკულატორი

☀️ ჯამური პანელები: 2700 W (2.7 kW)
⚡ რეკომენდებული ინვერტორი: ≥ 3.5 kW (Pure Sine)
🎛️ MPPT კონტროლერი: ≥ 100 A
🔋 აკუმულატორის ბანკი: ~ 330 Ah (24V-ზე)
🛡️ DC დაცვა & ავტომატი: 125A DC + SPD Type 2
🔌 AC გამომავალი ავტომატი: 20A C-ტიპი (220V)

☀️ მზის ელექტროსისტემების სრული სახელმძღვანელო

1. მზის პანელების ტიპები და შერჩევა

• მონოკრისტალური (Monocrystalline / PERC / N-Type): დამზადებულია ერთგვაროვანი სილიციუმისგან. აქვს ყველაზე მაღალი მარგი ქმედების კოეფიციენტი (20-23%) და კარგად მუშაობს მოღრუბლულ ამინდშიც.

• პოლიკრისტალური (Polycrystalline): შედარებით იაფია, ეფექტურობა 15-17%.

• ორმხრივი (Bifacial): ითვისებს სინათლეს ორივე მხრიდან, რაც ეფექტურობას 10-25%-ით ზრდის.

2. ინვერტორები და MPPT კონტროლერები

• Pure Sine Wave (სუფთა სინუსოიდა): აუცილებელია ყველა ტიპის ძრავისა და გაზის ქვაბისთვის!

• MPPT კონტროლერი: 30%-ით უფრო ეფექტურია PWM ტიპზე.
